## Setup
1. Clone the repo: `git clone https://github.com/your-username/ai-companion-chatbot.git`
2. Install dependencies: `npm install`
3. Set up `.env` with `DATABASE_URL`, `XAI_API_KEY`, `NEXTAUTH_URL`, `NEXTAUTH_SECRET`
4. Run: `npm run dev`
